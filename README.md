# School Management System

Spring Boot REST API project for managing School, Students and Teachers.

## Technologies
- Java 17+
- Spring Boot
- Maven
- REST API
- Postman (for testing)

## Project Structure
- controller
- entities
- main application class

## Endpoints

### GET
`/school` – Check server status  
`/school/info` – Get school data

### POST
`/school/specific` – Send student data via request parameter

### PUT
`/school/update` – Update school data

### DELETE
`/school/delete` – Delete by ID
